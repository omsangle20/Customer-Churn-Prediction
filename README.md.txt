# Capstone Project: Customer Churn Prediction in Telecom Industry

## Project Overview
This project focuses on predicting customer churn in the telecom industry, specifically targeting high-value customers in the Indian and Southeast Asian prepaid market.

## Data Description
The dataset contains customer usage data over four months, with key features related to voice calls, internet usage, and recharge amounts.

## Methodology
- Data Preprocessing: Cleaned and prepared the data for analysis.
- Feature Engineering: Created relevant features for modeling.
- Modeling: Employed Logistic Regression and used GridSearchCV for hyperparameter tuning.

## Results
The final model achieved an accuracy of 100% on the test set, indicating effective churn prediction.

## Conclusion
The analysis provides insights into customer behavior, with strategic recommendations for reducing churn among high-value customers.
